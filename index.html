Welcome

This project uses machine learning to predict flood-prone areas within GN divisions in the Gampaha district of Sri Lanka.

About the Project

Flooding is a recurring issue in Sri Lanka, especially in the Gampaha district. This study applies ML models like SVM, Random Forest, and XGBoost to historical data since 2016 for accurate flood area prediction. The goal is to assist decision-makers and emergency response teams with timely insights.

Methodology

We collected data from Sri Lanka’s Irrigation Department, Disaster Management Centre, and the Department of Meteorology. Key features used in the model are:

  • Distance from Attanagalu Oya
  • Water level readings of Attanagalu Oya
  • Rainfall data

Preprocessing was performed using Python and Pandas. We trained and evaluated three models (SVM, Random Forest, XGBoost) with the goal of predicting a binary outcome: 1 (Flood) or 0 (No Flood).

Results

Each model was evaluated for accuracy, precision, and recall. The model with the highest accuracy will be selected for final deployment. Visualizations include:

  • Confusion Matrix
  • ROC Curve
  • Flood risk maps based on predictions

Discussion

Our study demonstrates that ML models, especially tree-based algorithms like XGBoost and Random Forest, perform well in predicting flood risks. The primary challenges included data gaps and the need for real-time integration. Future improvements will include sensor-based inputs and integration with alert systems.
